//PostgreSQL + SQLAlchemy
pip install sqlalchemy alembic psycopg2-binary python-dotenv

-> SQLAlchemy → ORM (similar to Laravel Eloquent)
-> Alembic → Database migrations (similar to Laravel migrations)
-> psycopg2-binary → PostgreSQL driver
-> python-dotenv → Load environment variables from .env


//Verify
pip show sqlalchemy
pip show psycopg2-binary


// Why PostgreSQL?

-> Because most FastAPI production applications use it.

1) Database	        :-Recommended for
2) SQLite	        :-Learning and small projects
3) MySQL	        :-Many PHP/Laravel applications
4) PostgreSQL	    :-FastAPI, Django, enterprise Python projects

-> PostgreSQL also supports advanced features like JSON columns, arrays, and powerful indexing, which make it a common choice for Python backends.
